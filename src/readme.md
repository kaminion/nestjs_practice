# Architect
- module -> controller -> service


# entry point
main.ts

# module
한가지 역할을 하는 모듈

# controller
controller는 는 url을 가져오고 함수를 실행하는 역할임 (Router)
MVC 의 C 맞음

# service
실제로 function을 가짐, 비즈니스 로직 실행

데코레이터는 꾸며주는 함수와 클래스에 있어야함
